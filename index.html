Projects

Esp32 Spotify Dashboard

An ESP32 Spotify Dashboard is a hardware project that displays live Spotify track information—such as song title, artist, and playback status—in real time on an SSD1306 OLED screen using an ESP32 microcontroller, Python, and the Spotify API

Python C++ Esp32 Spotify Web-Api

Planetary Projectile Pathway

Planetary Projectile Pathway is an interactive simulator that visualizes 2D projectile motion across different planets, allowing users to customize gravity, launch angle, and wind conditions with real-time animated trajectories and instant physics calculations. Built for physics education and experimentation, it features a large, themed display area and a user-friendly Python/Tkinter interface for intuitive exploration.

Python Tkinter Python standard Math library

Sim Racing Dashboard for Assetto Corsa

The sim-racing-dashboard is an ESP32-based embedded system that delivers real-time F1-style telemetry from Assetto Corsa to a physical SSD1306 OLED display, showing live speed, gear, lap time, pit status, and car name for a true racing HUD experience. It combines C++ firmware for the display, a Python telemetry extractor, and serial communication to bridge virtual racing with a hands-on dashboard build

ESP32 microcontroller C++ Python pyaccsharedmemory
